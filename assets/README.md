# Assets

Esta pasta centraliza os recursos visuais e sonoros do jogo.

## Organização

- `imagens/`: imagens, sprites e elementos visuais.
- `sons/`: efeitos sonoros e músicas.
- `fontes/`: fontes tipográficas customizadas.

## Recursos utilizados

Nesta versão do projeto não foram utilizados assets externos.

Todos os elementos visuais (jogador, paredes, portas, chaves, saída e interface) são desenhados diretamente pelo código utilizando as funções da biblioteca Pygame.

Também não foram utilizadas músicas, efeitos sonoros ou fontes externas, sendo empregadas apenas as fontes padrão disponibilizadas pelo próprio Pygame.

## Boas práticas

- Manter arquivos organizados em suas respectivas pastas.
- Registrar a origem de qualquer asset externo adicionado futuramente.
- Utilizar formatos leves e amplamente suportados, como .png, .wav, .ogg e .ttf.
