# Código-fonte (`src`)

Esta pasta contém os módulos principais do jogo Lost Path.

## Arquivos

- `jogo.py`: tela inicial, loop principal, HUD, telas de vitória, derrota, pausa e controle geral da partida.
- `config.py`: configurações gerais, constantes, cores, tamanho da tela e velocidade do jogador.
- `mapa.py`: definição do labirinto, desenho do mapa, portas, chaves e saída.
- `player.py`: movimentação, colisão e desenho do jogador.
- `ranking.py`: gravação e leitura do ranking de jogadores em arquivo texto.

## Dica de evolução

Quando o projeto crescer, mantenha módulos pequenos e separados por responsabilidade.

## Organização

Cada módulo possui uma responsabilidade específica, facilitando a manutenção e futuras melhorias do projeto.
